# Dataset License and Attribution

## SA-MahaBERT Dataset

This dataset is derived from the L3Cube-MahaNews dataset developed by L3Cube Pune for Marathi Natural Language Processing research.

### Original Dataset

Source: L3Cube-MahaNews

Authors: L3Cube Pune Research Group

### Modifications

The original dataset was modified for sentiment analysis purposes.

The following changes were made:

* Removed the original news category/classification labels.
* Added sentiment annotations (Positive, Negative and Neutral).
* Processed and formatted the dataset for sentiment classification tasks.
* Generated training, validation, and testing splits during experimentation.
